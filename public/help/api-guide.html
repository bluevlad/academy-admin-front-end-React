Academy Admin API Guide

Backend REST API 구성 및 접속 가이드

1. URL 구조

백엔드는 /admin context-path 아래에서 모든 엔드포인트를 제공합니다.

기본 구조

https://academy.unmong.com/admin/api/{module}/{endpoint}
구분URL설명
Production 도메인https://academy.unmong.com통합 도메인
Context Path/admin관리자 서비스 컨텍스트
API Prefix/api각 컨트롤러의 RequestMapping
참고: 이전에 사용하던 admin.unmong.com:8080 호스트는 더 이상 사용하지 않습니다. 모든 서비스는 academy.unmong.com 도메인으로 통일되었습니다.

4. 프론트엔드 연동 설정

프론트엔드는 VITE_API_URL 환경변수로 API baseURL을 설정합니다.

Docker (Production)

# docker-compose.yml
environment:
  - VITE_API_URL=/admin/api

로컬 개발 (기본값)

// shared/api/client.js
const BASE_API = import.meta.env.VITE_API_URL || "http://academy.unmong.com:9001/admin/api";

API 호출 예시

// baseURL: /admin/api
// 호출: apiClient.get("/board/getBoardMngList")
// 최종 경로: /admin/api/board/getBoardMngList

6. Gateway (Nginx) 프록시 설정

메인 Gateway에서 academy.unmong.com 도메인으로 들어오는 요청을 각 서비스로 라우팅합니다.

필요한 프록시 룰

LocationProxy Target대상
/admin/api/http://localhost:9001/admin/api/Backend API
/admin/swagger-ui/http://localhost:9001/admin/swagger-ui/Swagger UI
/admin/v3/http://localhost:9001/admin/v3/OpenAPI Docs
/admin/http://localhost:4001/admin/Frontend (React)

Nginx 설정 예시

# Backend API + Swagger
location /admin/api/ {
    proxy_pass http://127.0.0.1:9001/admin/api/;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
}

location /admin/swagger-ui/ {
    proxy_pass http://127.0.0.1:9001/admin/swagger-ui/;
    proxy_set_header Host $host;
    proxy_set_header X-Forwarded-Proto $scheme;
}

location /admin/v3/ {
    proxy_pass http://127.0.0.1:9001/admin/v3/;
    proxy_set_header Host $host;
    proxy_set_header X-Forwarded-Proto $scheme;
}

# Frontend (React) - 가장 마지막에 배치
location /admin/ {
    proxy_pass http://127.0.0.1:4001/admin/;
    proxy_set_header Host $host;
    proxy_set_header X-Forwarded-Proto $scheme;
}
순서 중요: Nginx location 블록은 /admin/api/, /admin/swagger-ui/, /admin/v3/를 /admin/보다 먼저 선언해야 합니다.
