Academy Admin Console Guide

관리자 콘솔 서비스 구성 및 운영 가이드

1. 서비스 개요

Academy Admin은 소규모 온라인 학원의 관리 업무를 위한 통합 관리 시스템입니다.

구성기술 스택
BackendSpring Boot 3.2 + MyBatis + MySQL (Java 17)
FrontendReact 18 + Vite 6 + MUI 6
인증JWT + Google OAuth 2.0

2. 접속 정보

Production

서비스URL
관리자 콘솔https://academy.unmong.com/admin/
API 문서 (Swagger)https://academy.unmong.com/admin/swagger-ui/index.html

로컬 개발

서비스URL
Frontend (Vite)http://localhost:3000/admin/
Backend APIhttp://localhost:9001/admin/api/
Swagger UIhttp://localhost:9001/admin/swagger-ui/index.html
도메인 통일: 모든 서비스는 academy.unmong.com 도메인 아래 /admin/ 컨텍스트에서 제공됩니다.

3. Docker 구성

컨테이너 구성

컨테이너이미지포트Context Path
academy-admin-back-endSpring Boot JAR9001/admin
academy-admin-frontVite Dev Server4001 → 3000/admin/

실행 명령어

# 전체 서비스 빌드 및 실행
docker compose up -d --build

# 백엔드만 재빌드
docker compose up -d --build academy-admin-backend

# 로그 확인
docker compose logs -f academy-admin-backend

# 서비스 중지
docker compose down

환경변수 (docker-compose.yml)

변수기본값설명
SERVER_PORT9001백엔드 서버 포트
DB_URLjdbc:mysql://172.30.1.72:3306/acm_basicDB 접속 URL
VITE_API_URL/admin/api프론트엔드 API baseURL
GOOGLE_CLIENT_ID-Google OAuth Client ID
SUPER_ADMIN_EMAILS-슈퍼 관리자 이메일 목록

4. 프로젝트 구조

academy-admin/                  # 워크스페이스 (Git 저장소 아님)
├── backend/                    # Spring Boot (독립 Git 저장소)
│   ├── src/main/java/com/academy/
│   │   ├── auth/               # 인증/권한
│   │   ├── board/              # 게시판
│   │   ├── member/             # 회원
│   │   ├── lecture/            # 강의 (Category, Form, Kind 등)
│   │   ├── book/               # 교재
│   │   ├── exam/               # 시험
│   │   ├── config/             # OpenAPI, WebMvc 설정
│   │   ├── common/             # 공통 유틸리티
│   │   └── mapper/             # MyBatis Mapper 인터페이스
│   └── src/main/resources/
│       ├── application.yml     # 서버 설정 (context-path: /admin)
│       ├── mapper/             # MyBatis XML
│       └── static/help/        # 도움말 페이지
├── frontend/                   # React (독립 Git 저장소)
│   ├── src/
│   │   ├── api/                # API 모듈 (Axios)
│   │   ├── shared/api/client.js  # 통합 API 클라이언트
│   │   └── constants/index.js  # 상수 정의
│   └── vite.config.js          # base: "/admin/"
└── docker-compose.yml          # 통합 서비스 관리
